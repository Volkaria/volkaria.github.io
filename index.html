Discord js Coding

I learnt javascript because I wanted to make a discord bot to run discord servers.

  • Example Commands

    Some Commands that work right now are ~info, ~serverinfo, ~botinfo, and ~help. Info will display an embedded message showing when that user joined and when the account was created. Server Info will display the Discord Server name, the amount of people in the server, and when the server was created. Bot Info will display the Bot Name and when the bot was created. Help will display the upcoming bot commands and show what the bot is capable of.
